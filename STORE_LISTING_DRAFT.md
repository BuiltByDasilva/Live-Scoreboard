# Chrome Web Store Listing Draft

## Name

Live Scoreboard

## Short description

Unofficial live football scores, match watchlist reminders, and fan-inspired skins for the 2026 tournament.

## Full description

Live Scoreboard is a lightweight browser companion for following the 2026 international football tournament while you browse.

Pin a match and its live score stays visible on the Chrome toolbar while you work. Open the side panel for all scores, the complete schedule, kickoff times, and watched matches. Add games to your watchlist to receive reminder notifications before kickoff. Personalize the panel with original country-inspired skins.

Built for focus:

- Live scoreboard side panel
- Toolbar score badge
- Upcoming match watchlist
- Kickoff reminders
- Local settings storage
- Fan-inspired team color skins
- No page-reading permissions

This extension is unofficial and is not affiliated with, endorsed by, or sponsored by any tournament organizer, governing body, team, federation, or broadcaster. Team and country names are used only for factual identification. Skin packs use original color themes and do not include official logos, crests, jerseys, marks, or protected artwork.

## Privacy summary

Live Scoreboard stores preferences and the last score snapshot locally using Chrome storage. It does not read browsing history, page content, cookies, or user-entered website data. Network access is limited to the declared live-score endpoint.
