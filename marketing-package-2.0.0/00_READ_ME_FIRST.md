# Live Scoreboard 2.0.0 Marketing Package

This folder contains the Chrome Web Store release materials for Live Scoreboard version `2.0.0`.

## Upload First

- `chrome-web-store-upload/live-scoreboard-extension.zip`

This is the installable Chrome extension package to upload in the Chrome Web Store dashboard.

## Listing Copy

- `listing-copy/STORE_LISTING_DRAFT.md`
- `listing-copy/PRIVACY_POLICY.md`
- `listing-copy/CHROME_WEB_STORE_PRIVACY_DISCLOSURE.md`
- `listing-copy/RELEASE_2.0.0.md`
- `listing-copy/README.md`

Publish the privacy policy at a public URL before submitting the listing.

## Store Assets

- Icon: `store-assets/icon/store-icon-128.png`
- Global screenshots: `store-assets/screenshots-global/`
- Localized screenshots: `store-assets/screenshots-localized/`
- Small promo tile: `store-assets/promo-tiles/small-promo-tile-440x280.jpg`
- Marquee promo tile: `store-assets/promo-tiles/marquee-promo-1400x560.jpg`

## Promo Video

- `video/live-scoreboard-promo.mp4`

Upload the MP4 to YouTube and paste the YouTube URL into the Chrome Web Store listing if you want to use a promo video.

## Final Dashboard Checks

1. Confirm version `2.0.0` appears after uploading the ZIP.
2. Paste the short and full descriptions from `STORE_LISTING_DRAFT.md`.
3. Add the public privacy policy URL.
4. Answer the privacy questionnaire using `CHROME_WEB_STORE_PRIVACY_DISCLOSURE.md`.
5. Upload the icon, screenshots, promo tiles, and promo video URL.
6. Confirm the listing says the app is unofficial and free to use.
